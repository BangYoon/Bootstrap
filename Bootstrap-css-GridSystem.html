Grid System 특성

  • 1. container 안에 row와 column이 있어야 한다. container은 내용의 배치과 패딩을 제어한다.
  • 2. container 안의 row가 column을 제어한다.
  • 3. 같은 row 내의 column들은 같은 패딩 값을 기본으로 가진다.
  • 4. column은 총 12개로 구성된다.
  • 5. 기본 패딩을 없애기 위해서는 row를 나누거나 속성을 사용한다.
  • 6. 반응형 웹에는 5개의 grid breakpoint가 존재한다. extra small, small, medium, large, extra large
  • 7. 기본으로 가장 작은 small grid breakpoint에 맞춰진다.
